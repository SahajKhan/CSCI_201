This Program is designed to help people shoot for the correct calories per day.
The program will have a file for username and passwords for login in a local file.
Each user will have their own file that stores data that has been inputed and suggests
calories based on the information provided. As you progress, you may update this information,
the old data will be kept and can be used for tracking purposes.

##################################################################################
User_Credentials - File that saves USER ID and Passwords for login.

The format is as follows:

UserID1 Password
UserID2 Password
etc.

##################################################################################
Each User will have a file that tracks current weight and calories. 
Based on the amount of weight to lose, the user will be recommended a
certain amount of calories to consume per day.

The file name will be the User ID.
The format is as follows:

NAME AGE HEIGHT GENDER

DATE_OF_INPUT
CURRENT_WEIGHT MONTHS_TO_LOSE_WEIGHT GOAL_WEIGHT CALORIES_PER_DAY

DATE_OF_INPUT
CURRENT_WEIGHT MONTHS_TO_LOSE_WEIGHT GOAL_WEIGHT CALORIES_PER_DAY
